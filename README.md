# XmlParserToPaint 
# 基于Xml格式的图形解析


**开发环境：Windows** 

**主要内容：利用qt开发界面，解析xml文件，生成各类复杂图形，支持图形变换，缩放，平移**

**技术难点：xml文件的解析，弧形路径的生成，公式的嵌套分析**

**经历/收获：从零开始学习了qt的各类组件，信号槽机制等，初步了解qt开发图形界面，提高了自己阅读英文文档的能力，调试程序的技巧，以及增强了对xml格式文件的了解**
